# Fruiting Chamber Calculator

A calculator for sizing mushroom fruiting chamber setups based on target weekly yield.

## Features

- Calculate blocks needed per batch from target weekly harvest
- Size chamber (grow tent) requirements
- FAE (fresh air exchange) and fan sizing
- Fogger selection and duty cycle estimates
- Support for 5lb and 10lb substrate blocks
- Adjustable biological efficiency (BE%)

## Usage

Open `fruiting-chamber-calculator.html` in a browser. No build step or dependencies required.

## Calculations

The calculator uses a 4-week batch cycle:
- 3 weeks production
- 1 week cleaning/restart

Key formula: `Yield per block = Block weight × 40% dry fraction × BE%`
